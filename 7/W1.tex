\section*{Постановка задачи}
Вопрос №44: Найти полином Лагранжа, который приближает функцию $f\left( x \right) = {x^3}$, используя узлы ${x_0} = 1$, ${x_1} = 2$.

\section*{Решение задачи}
Найдем формулу интерполяции для $f\left( x \right) = {x^3}$ имеющей следующие значения: 
$$
\begin{array}{l}
{x_0} = 1, f\left( {{x_0}} \right) = 1\\
{x_1} = 2, f\left( {{x_0}} \right) = 8
\end{array}
$$
Воспользуясь формулой Лагранжа, получим:
$$
L\left( x \right) = \frac{{\left( {x - 2} \right)}}{{\left( {1 - 2} \right)}} + \frac{{8\left( {x - 1} \right)}}{{\left( {2 - 1} \right)}} = 8x - 8 - x + 2 = 7x - 6
$$
Проверяем:
$$
\begin{array}{l}
L\left( {{x_0}} \right) = L\left( 1 \right) = 7 - 6 = 1\\
L\left( {{x_1}} \right) = L\left( 2 \right) = 14 - 6 = 8
\end{array}
$$
Тоесть: $L\left( {{x_0}} \right) = f\left( {{x_0}} \right)$ как и $L\left( {{x_1}} \right) = f\left( {{x_1}} \right)$. Следовательно, полученное решение — верно.
