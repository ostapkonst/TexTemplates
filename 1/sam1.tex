\section{Поиск обратной матрицы Методом Гаусса}

\subsection{Постановка задачи}
Дана матрица квадратная невырожденная матрица\footnote{Квадратная матрица обратима тогда и только тогда, когда она невырожденная, то есть её определитель не равен нулю. Для неквадратных матриц и вырожденных матриц обратных матриц не существует.} вида:
$$
\left( {\begin{array}{*{20}{c}}
{{a_{11}}}& \ldots &{{a_{1n}}}\\
 \vdots & \ddots & \vdots \\
{{a_{n1}}}& \cdots &{{a_{nn}}}
\end{array}} \right).
$$
Требуется найти обратную ей матрицу\footnote{Обратная матрица — такая матрица ${\rm A}^{ - 1}$, при умножении на которую, исходная матрица ${\rm A}$ даёт в результате единичную матрицу ${\rm E}$.}, используя метод Гаусса.

\subsection{Алгоритм решения}
Напишем расширенную матрицу, в левой части которой находится исходная матрица ${\rm A}$, а в правой единичная. Применяя метод Гаусса, последовательно будем приводить матрицу ${\rm A}$ (левую часть расширенной матрицы) к единичной матрице. Причем совершенные преобразование мы будем применять ко всей расширенной матрице. Приведя левую часть расширенной матрицы к единичной, правая часть будет являться обратной матрицей к исходной.

\subsection{Пример решения}
Найти обратную матрицу ${\rm A}^{ - 1}$ методом Гаусса, если 
$$
{\rm A} = \left( {\begin{array}{*{20}{c}}
3&5&7\\
9&{11}&{13}\\
{17}&{19}&{23}
\end{array}} \right).
$$
Сначала проверим не является ли матрица ${\rm A}$ вырожденной (сингулярной), для чего вычислим её определить:
$$
\begin{array}{l}
\det {\rm A} = \left| {\begin{array}{*{20}{c}}
3&5&7\\
9&{11}&{13}\\
{17}&{19}&{23}
\end{array}} \right| = \\
 = 3 \times 11 \times 23 + 5 \times 13 \times 17 + 7 \times 9 \times 19 - \\
 - \left( {7 \times 11 \times 17 + 5 \times 9 \times 23 + 3 \times 13 \times 19} \right) = \\
 = 759 + 1105 + 1197 - \left( {1309 + 1035 + 741} \right) = \\
 = 3061 - 3085 =  - 24.
\end{array}
$$
Так как определитель ~не равен нулю, то, следовательно, матрица ${\rm A}$ невырожденная (регулярная) и существует ей обратная матрица ${\rm A}^{ - 1}$. Далее, по вышеописанному алгоритму, найдём единичную матрицу:
$$
\begin{array}{l}
\left( {\begin{array}{*{20}{c}}
3&5&7&\vline& 1&0&0\\
9&{11}&{13}&\vline& 0&1&0\\
{17}&{19}&{23}&\vline& 0&0&1
\end{array}} \right) \Rightarrow \\
 \Rightarrow \left( {\begin{array}{*{20}{c}}
3&5&7&\vline& 1&0&0\\
0&{ - 4}&{ - 8}&\vline& { - 3}&1&0\\
{17}&{19}&{23}&\vline& 0&0&1
\end{array}} \right) \Rightarrow \\
 \Rightarrow \left( {\begin{array}{*{20}{c}}
3&5&7&\vline& 1&0&0\\
0&{ - 4}&{ - 8}&\vline& { - 3}&1&0\\
0&{ - \frac{{28}}{3}}&{ - \frac{{50}}{3}}&\vline& { - \frac{{17}}{3}}&0&1
\end{array}} \right) \Rightarrow \\
 \Rightarrow \left( {\begin{array}{*{20}{c}}
3&0&{ - 3}&\vline& { - \frac{{11}}{4}}&{\frac{5}{4}}&0\\
0&{ - 4}&{ - 8}&\vline& { - 3}&1&0\\
0&{ - \frac{{28}}{3}}&{ - \frac{{50}}{3}}&\vline& { - \frac{{17}}{3}}&0&1
\end{array}} \right) \Rightarrow \\
 \Rightarrow \left( {\begin{array}{*{20}{c}}
3&0&{ - 3}&\vline& { - \frac{{11}}{4}}&{\frac{5}{4}}&0\\
0&{ - 4}&{ - 8}&\vline& { - 3}&1&0\\
0&0&2&\vline& {\frac{4}{3}}&{ - \frac{7}{3}}&1
\end{array}} \right) \Rightarrow \\
 \Rightarrow \left( {\begin{array}{*{20}{c}}
3&0&0&\vline& { - \frac{3}{4}}&{ - \frac{9}{4}}&{\frac{3}{2}}\\
0&{ - 4}&{ - 8}&\vline& { - 3}&1&0\\
0&0&2&\vline& {\frac{4}{3}}&{ - \frac{7}{3}}&1
\end{array}} \right) \Rightarrow \\
 \Rightarrow \left( {\begin{array}{*{20}{c}}
3&0&0&\vline& { - \frac{3}{4}}&{ - \frac{9}{4}}&{\frac{3}{2}}\\
0&{ - 4}&0&\vline& {\frac{7}{3}}&{ - \frac{{25}}{3}}&4\\
0&0&2&\vline& {\frac{4}{3}}&{ - \frac{7}{3}}&1
\end{array}} \right) \Rightarrow \\
 \Rightarrow \left( {\begin{array}{*{20}{c}}
1&0&0&\vline& { - \frac{1}{4}}&{ - \frac{3}{4}}&{\frac{1}{2}}\\
0&1&0&\vline& { - \frac{7}{{12}}}&{\frac{{25}}{{12}}}&{ - 1}\\
0&0&1&\vline& {\frac{2}{3}}&{ - \frac{7}{6}}&{\frac{1}{2}}
\end{array}} \right).
\end{array}
$$
Итак, окончательно имеем:
$$
{{\rm A}^{ - 1}} = \left( {\begin{array}{*{20}{c}}
{ - \frac{1}{4}}&{ - \frac{3}{4}}&{\frac{1}{2}}\\
{ - \frac{7}{{12}}}&{\frac{{25}}{{12}}}&{ - 1}\\
{\frac{2}{3}}&{ - \frac{7}{6}}&{\frac{1}{2}}
\end{array}} \right).
$$

\subsection{Решение MATLAB}
Порядок решения задачи в MATLAB следующий:
\begin{itemize}
	\item сформировать матрицу коэффициентов ${\rm A}$ и вектор свободных членов ${\rm B}$ заданной системы;
	\item количество формирований вектора ${\rm B}$ происходит в соответствии размерности матрицы ${\rm A}$;
	\item с каждым формированием ${\rm B}$ - формировать расширенную матрицу системы, ${\rm A}$ объединив и ${\rm B}$;
	\item решить полученную систему линейных уравнений методом Гаусса;
	\item полученный вектор будет являться столбцом матрицы ${\rm A}^{ - 1}$;
\end{itemize}

\subsection{Подсчёт невязки}
Невязка — это ошибка (погрешность) в результате вычислений.
\begin{lstlisting}
	Размерность матрицы
		n = 5;
	Генерация исходной матрицы
		A = rand(n);
	Обратная матрица
		B = rinv(A);
	Подсчёт невязки
		Discrepancynorm = (A * B - eye(n));
\end{lstlisting} 

\section{Поиск обратной матрицы Метод окаймления}

\subsection{Постановка задачи}
Дана матрица квадратная невырожденная матрица\footnote{Квадратная матрица обратима тогда и только тогда, когда она невырожденная, то есть её определитель не равен нулю. Для неквадратных матриц и вырожденных матриц обратных матриц не существует.} вида:
$$
\left( {\begin{array}{*{20}{c}}
{{a_{11}}}& \ldots &{{a_{1n}}}\\
 \vdots & \ddots & \vdots \\
{{a_{n1}}}& \cdots &{{a_{nn}}}
\end{array}} \right).
$$
Требуется найти обратную ей матрицу\footnote{Обратная матрица — такая матрица ${\rm A}^{ - 1}$, при умножении на которую, исходная матрица ${\rm A}$ даёт в результате единичную матрицу ${\rm E}$.}, используя метод окаймления.

\subsection{Алгоритм решения}
Пусть матрица ${\rm A} = \left\| {{a_{ij}}} \right\|,i,j = 1,2,...,n$, невырожденная. Обозначим через ${{\rm A}_k}$ ее левую верхнюю часть, т. е. ${{\rm A}_k} = \left\| {{a_{ij}}} \right\|,i,j = 1,2,...,k$. Матрицы ${{\rm A}_k},k = 1,2,...,n$, невырожденные. Метод окаймления состоит в следую­щем: ищется матрица, обратная к
$$
{{\rm A}_k} = \left\| {\begin{array}{*{20}{c}}
{{{\rm A}_{k - 1}}}&{{u_k}}\\
{v_k^T}&{{a_k}}
\end{array}} \right\|,k > 1\,,{{\rm A}_1} = {a_1} = {a_{11}},{u_1} = 0,v_1^T = 0,
$$
в форме
$$
{\rm A}_k^{ - 1} = \left\| {\begin{array}{*{20}{c}}
{{{\rm P}_{k - 1}}}&{{r_k}}\\
{q_k^T}&{{b_k}}
\end{array}} \right\|.
$$
$$
{b_k} = {\left( {{a_k} - v_k^T{\rm A}_{k - 1}^{ - 1}{u_k}} \right)^{ - 1}},{r_k} =  - {b_k}{\rm A}_{k - 1}^{ - 1}{u_k},q_k^T =  - {b_k}v_k^T{\rm A}_{k - 1}^{ - 1},
$$
$$
{{\rm P}_{k - 1}} = {\rm A}_{k - 1}^{ - 1} + {b_k}{\rm A}_{k - 1}^{ - 1}{u_k}v_k^T{\rm A}_{k - 1}^{ - 1}.
$$

\subsection{Решение MATLAB}
Порядок решения задачи в MATLAB следующий:
\begin{itemize}
	\item количество шагов будет совпадать с размерностью матрицы ${\rm A}$;
	\item формировать на каждом шаге рабочую матрицу ${\rm B}$;
	\item с каждым формированием ${\rm B}$, совершать действия в соответствии с текущим шагом;
	\item если шаг является последним, то формирование обратной матрицы завершено;
	\item иначе, выполнять действия над матрицей, по формулам окаймления;
\end{itemize}

\subsection{Подсчёт невязки}
Невязка — это ошибка (погрешность) в результате вычислений.
\begin{lstlisting}

		n = 5;
	Генерация исходной матрицы
		A = rand(n);
	Обратная матрица
		B = edging(A);
	Подсчёт невязки
		Discrepancynorm = (A * B - eye(n));
\end{lstlisting}
